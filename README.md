# Развертывание ClickHouse в Kubernetes

## Описание
- Helm-чарт для разворачивания ClickHouse с параметрами версии и списком пользователей.
- Поддерживается настройка конфигураций через ConfigMap.
- Возможность поведения выбора версии образа ClickHouse.
- Автоматическое создание пользователей с их паролями.

## Использование
1. Указать желаемую версию в values.yaml в секции image.tag
2. Определить пользователей и пароли в секции users
3. Установить чарты командой `helm install`
4. Логи и данные приложения находятся внутри контейнера

## Требования
- Kubernetes кластер (1.20+)
- Helm 3+
